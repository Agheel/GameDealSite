🎮 Game Deals Aggregator (게임 핫딜 모아보기)
여러 게임 판매 사이트(Steam, Epic Games, GMG)의 할인 정보를 실시간으로 수집하여 한눈에 보여주는 웹 애플리케이션입니다. React와 FastAPI를 사용하여 구축되었으며, Firebase를 통해 데이터를 관리합니다.

✨ 주요 기능 (Key Features)
🔍 멀티 플랫폼 할인 정보: Steam, Epic Games, Green Man Gaming의 실시간 할인 정보를 통합 제공

🇰🇷 자동 한글화: deep-translator를 활용하여 영어로 된 게임 설명을 자동으로 한글 번역

📱 반응형 그리드 UI: 화면 크기에 따라 최적화되는 카드형 레이아웃 (스팀 스타일)

🎬 시네마틱 모달: 게임 클릭 시 원본 비율을 유지하는 몰입감 있는 상세 정보 창 (Cinema Mode)

⚡ 강력한 필터링: 할인율순, 메타스코어순, 가격순 정렬 및 장르별 필터 제공

🌙 다크 모드: 눈이 편안한 다크 모드 기본 지원
